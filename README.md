📘 SemSnap

SemSnap is a smart CGPA calculator built for VTU students that automatically extracts subject codes and marks directly from result PDFs and calculates CGPA instantly.

No manual entry. No messy spreadsheets. Just upload → snap → calculate.

🚀 What It Does

📄 Upload semester result PDFs

🔍 Automatically extracts subject codes & marks

🧠 Maps them with official scheme credits

📊 Calculates accurate CGPA instantly

🎓 Supports multiple schemes (2022, 2025, etc.)

⚙️ Tech Stack

Python (Flask) – Backend

HTML – Frontend

PyMuPDF / OCR – PDF parsing

JSON-based Scheme System – Credit mapping

Modular CGPA Engine

💡 Why SemSnap?

VTU result sheets can be messy. Manual CGPA calculation is boring and error-prone.

SemSnap simplifies everything into:

Upload → Parse → Map → Calculate

Built by a student, for students.
